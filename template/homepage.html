This is my Indie Web site...

Meaning I hacked it together myself - both the html content, and the software it runs on. I've only just started IndieWebbing, so it still doesn't support a lot of types of interactions. Please be patient! :)

My current main project is an unhosted web app called Meute. It leverages remotestorage.js, sockethub, and 5apps to create an email+irc+POSSE blogging tool in the browser. Other projects I'm involved in include Terms of Service; Didn't Read and Opentabs.
