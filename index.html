v0.6.0

Meute is still very basic, but it can already help you build an app that combines remotestorage for per-user data storage and sockethub v0.2 for relaying messages from the browser to the outside world. The code is on github.

NOTE: You need version 0.2 of sockethub, version 0.1 will not work!
