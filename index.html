v0.2.0

meute.js is still very basic, but it can already help you build an app that combines remotestorage for per-user data storage and sockethub for relaying messages from the browser to the outside world. The code is on github.

The 0.2 API still follows the sockethub syntax very closely and is as follows: 

      meute.setMasterPassword('master');
      meute.addAccount('sockethub', {
        host: 'michielbdejong.com',
        ssl: true, tls: true, port: 10550, path: '',
        register: { secret: '1234567890' }
      });
      meute.addAccount('irc', {
        credentials: {
          'nick': {
            nick: 'nick',
            server: 'irc.freenode.net',
            password: '',
            channels: ['#meute'],
            actor: {
              address: 'nick',
              name: 'nick'
            }
          }
        }
      });
      meute.send({
        platform: 'irc',
        verb: 'send',
        actor: { address: 'nick', name: 'nick' },
        target: [{
          address: '#meute'
        }],
        object: {
          text: 'hi'
        }
      });

The credentials you give in the addAccount calls will be encrypted with the master password using sjcl and stored in the respective remotestorage modules.
