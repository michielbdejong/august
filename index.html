There is not much to see here yet. Meute is an attempt to build a polyglot client for the federated social web, in an unhhosted web app, using technologies like remotestorage for per-user data storage, sockethub for relaying messages from the browser to the outside world, and useraddress for user search. It will initially support human-readable email and irc messages, but plans also exist to add support for comment threads on github issue, Facebook chat, and opentabs. The code is on github.
